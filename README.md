# Ean-curso\
Tema elegido: Análisis y proyección de ventas mensuales y gastos operativos.
Fuente de datos: Dataset propio simulado para el ejercicio práctico (12 filas).
Funciones o herramientas de IA usadas: Herramienta "Analizar datos" (Ideas) nativa de Excel para el reconocimiento de patrones y generación automática de visualizaciones.
Conclusiones del análisis: La IA permitió identificar rápidamente una correlación positiva y una tendencia al alza en las ventas durante el segundo semestre del año. El cruce de datos sugiere que el último trimestre es el más rentable, lo cual ayuda a tomar decisiones sobre dónde concentrar el presupuesto de marketing para el próximo año.
